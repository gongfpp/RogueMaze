ROGUE MAZE — ALPHA PLAYTEST

Thank you for testing an unfinished game.

START
1. Extract the entire ZIP to a normal folder.
2. Run RogueMaze.exe.
3. Windows SmartScreen may show an unsigned-app warning. Confirm the file came from the project team and compare SHA256SUMS.txt before continuing. Do not disable antivirus globally.

CONTROLS
- Click or drag one of the four road cards onto a green cell.
- R or right-click rotates the selected card.
- Number keys 1–4 select cards. Space pauses.
- The runner starts automatically after four seconds.
- Between nodes, choose ADD, ARMOR, or REMOVE.

TEST TASK
Please try to finish one expedition without asking for the solution. Thinking aloud is helpful. It is fine to fail or stop.

BUILD AND PRIVACY
- Pause or finish a run to see the version, platform, and short Git commit.
- The game does not upload play data and has no account.
- The facilitator should record a participant code, not your real name.

IF SOMETHING BREAKS
Tell the facilitator what you did immediately before the problem. Include the Build line from the game or BUILD.txt and the two summary lines from the result screen.
